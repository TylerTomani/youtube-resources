01

Add 3 let variables for canasRect,clickX, and clickY 

let canavsRect
let clickX
let clickY

02

 Use the above 3 variables,in the mousdown addEventListener, to the clicked x and y position on the canvas.

 Also, set the

  • keys.up.pressed
  • keys.down.pressed .. etc
to false in the mouseup eventListener We are using keys so We can also change it back to keyboard movement 

canvas.addEventListener('mousedown', e => {
 canavsRect = canvas.getBoundingClientRect();
 clickX = e.clientX - canavsRect.left;
 clickY = e.clientY - canavsRect.top;
}) ;
canvas.addEventListener('mouseup', e => {
 keys.up.pressed =  false
 keys.right.pressed = false
 keys.down.pressed = false
 keys.left.pressed = false
})

03

Now assigne the keys.pressed event within the mousedown event based on the clickX and clickY variables
