02

Instead of adding methods to individual objects add the method to a class and create instances of that class. 

class Person {
    talk(){
        return 'talking'
    }
}
const me = new Person()
const you = new Person()
me.talk()
you.talk()

Now the method "inherits" from the talk() method from the Person class
