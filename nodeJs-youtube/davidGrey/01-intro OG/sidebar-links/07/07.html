7 2 other ways, create objects that inherit

Object.create() and Object. setPrototypeOf(instance,parent class)

01

Create an instance of an object {} with Object.create() 

const person = {
    talk(){
        return 'Talking'
    }
}
const me = Object.create(person)
me.talk()

01

Object.setPrototypeOf(instance, parent class) 

const person = {
    talk(){
        return 'Talking'
    }
}
const me = {}
Object.setPrototypeOf(me,person)
me.talk()
