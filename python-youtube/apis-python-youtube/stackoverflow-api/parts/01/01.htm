This is the basic script to create the youtube api service. The function dir() will return the methods and static attributes of a api.

api_key = 'AIxxxxxxx'

Part 6 - basic script - full script

01

Use the info and arguments from the previous step to request statitics from schafer5 

from googleapiclient.discovery import build

api_key = 'Axxxxxx'

service = build('youtube', 'v3',developerKey=api_key)

print(dir(service))
request = service.channels().list(
part='statistics',
forUsername='schafer5'
)
response = request.execute()
print(response)
